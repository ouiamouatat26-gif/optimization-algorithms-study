# optimization-algorithms-study

Ce projet propose une analyse technique et comparative de différents algorithmes d'optimisation utilisés dans le domaine du Machine Learning. L'objectif est d'observer et de quantifier le comportement de ces algorithmes (convergence, stabilité, performance) à travers une mise en pratique concrète.

##  Objectifs du projet
- Étudier le comportement des algorithmes d'optimisation (ex: SGD, Adam).
- Comparer l'impact des hyperparamètres sur la vitesse de convergence.
- Visualiser les résultats à l'aide d'outils graphiques.

##  Technologies utilisées
- Langage :Python
- Manipulation de données :Pandas
- Visualisation :Matplotlib
- Environnement :Jupyter Notebook
